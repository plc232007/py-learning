This repository is dedicated to my Python learning journey.

Here you will find exercises, examples, and small scripts created while studying the fundamentals of Python programming. The goal is to build a solid understanding of logic, syntax, and problem-solving using Python.

## 📁 Structure

├── estudos/               # Aulas e anotações
│   ├── basico/            # Aulas 1–32 + anotações
│   └── intermediario/     # Funções, closures, dicts, escopo...
